# My_TODOLIST
TP To_Do_List cours de Développement WEB B1

-- Installation --
*Git pull dans votre dossier xampp/htdocs

*Dans 'config/' dupliquez le fichier 'config.example.php' et le renommer 'config.php' puis dans ce dernier, 
remplissez vos propres valeurs pour utilisateur et mot de passe.

*Sur Xampp, lancez Apache et Mysql

*Sur phpmyadmin, dans l'onglet SQL (à la racine, hors de toute BDD déjà créé), copier/coller le contenu du fichier database.sql pour créer la base de donnée complète avec les données de test déjà insérées.

-- Lancement --
*Dans votre navigateur allez ici : http://localhost/my_todolist/
